# Canvas Video Assignment

This project is a simple implementation of a canvas-based video/image renderer, completed as part of the internship assignment.

## Functionality
- Plays two scenes sequentially according to the provided requirements.
- Implements `typing` and `blink` text animations using native Canvas API.
- Supports play/pause functionality via clicking on the canvas.

## How to run
1. Ensure you have a local web server (e.g., Live Server in VS Code).
2. Open `videoincanvas.html` in your browser via the server.
