# Canvas Video Assignment

This project is a simple implementation of a canvas-based video/image renderer, completed as part of the internship assignment.

## Functionality
- Plays two scenes sequentially according to the provided requirements.
- Implements `typing` and `blink` text animations using native Canvas API.
- Supports play/pause functionality via clicking on the canvas.

## Notes
- This implementation focuses on core logic using Vanilla JS.
- As an intern with a primary focus on backend development, I have prioritized core functionality and functional logic in this frontend task.
- Known limitation: External media URLs (like GettyImages) may be subject to CORS restrictions in some browser environments. I am aware of this and understand that in a production environment, media assets should be handled via a proxy or CDN-enabled server.

## How to run
1. Ensure you have a local web server (e.g., Live Server in VS Code).
2. Open `videoincanvas.html` in your browser via the server.
